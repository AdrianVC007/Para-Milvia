# Para-Milvia

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Compartir por GitHub Pages

Este proyecto ya está configurado para funcionar en GitHub Pages del repositorio `Para-Milvia`.

1. Sube los cambios a GitHub.
2. Ve a **Settings → Pages**.
3. En **Build and deployment**, elige **Source: GitHub Actions**.
4. Publica la carpeta `dist/` (con tu workflow) o usa un action de Vite para deploy.

> Importante: usa el enlace de Pages (`https://<usuario>.github.io/Para-Milvia/`) y no el enlace del repositorio.
